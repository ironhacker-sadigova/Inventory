# Inventory

## Installation

Install [composer](http://getcomposer.org/doc/00-intro.md) and run:

```bash
$ composer install
```

## Run

Install [Virtual Box](https://www.virtualbox.org/wiki/Downloads) and [Vagrant](http://www.vagrantup.com/):

```bash
$ vagrant up
```

SSH to the virtual machine:

```bash
$ vagrant ssh
```

Destroy the VM:

```bash
$ vagrant destroy
```
